# Spring JDBC Customers

Backend application built with Spring Framework that demonstrates CRUD operations using JDBC.

## Tech Stack
- Java
- Spring Framework
- Spring JDBC (JdbcTemplate)
- SQL
- Maven

## Features
- Create, read, update, and delete customers
- REST API endpoints
- DAO pattern implementation
- RowMapper for mapping database rows
- Custom exception handling
- Clean layered architecture

## Project Structure
- Controller — handles HTTP requests
- DAO / Repository — database access layer
- Model — domain objects
- Exception handling

## Purpose
This project was created as a learning example to practice backend development with Spring and JDBC.
